## Testing Scenarios

### Approval Workflow Testing
1. Login as `procurement@accordo.ai`
2. Create requisition > $10,000 (requires Manager approval)
3. Login as `manager@accordo.ai` to approve
4. For requisitions > $50,000, `director@accordo.ai` must approve
5. For requisitions > $500,000, `vp@accordo.ai` must approve

### Negotiation Testing
1. Login as procurement user
2. Create deal with vendor (e.g., `sales@techsupply.com`)
3. Switch to vendor portal and login as vendor
4. Exchange offers and see L1/L2/L3 ranking

### Multi-Vendor Comparison
1. Create requisition with 3+ vendors
2. Complete negotiations with each vendor
3. View comparison page with L1/L2/L3 rankings
4. Select winner and generate PO
